# TypeORM Maintainers

This repo contains information and resources for the TypeORM maintainers.

## Background

TypeORM is one of the most popular ORM libraries for the Node.js ecosystem. Originally created by [pleerock](https://github.com/pleerock) and [AlexMesser](https://github.com/AlexMesser), in late 2024 maintenance was passed to [David Höck](https://github.com/dlhck) and [Michael Bromley](https://github.com/michaelbromley) who then put together the current maintainer team.
